# shea_data

这是一个个人知识库与学习笔记的备份仓库，主要用于存储和同步我在学习计算机科学与软件开发过程中的总结、思考与资料。

## 📖 仓库简介

这个仓库是我的“第二大脑”，内容完全使用 [Obsidian](https://obsidian.md/) 进行编辑和管理。它记录了我的学习轨迹，涵盖了从基础知识到具体技术问题的多个方面。

## 📂 内容结构

仓库的主要目录和文件如下：

*   **`LLM/`**: 关于大语言模型相关的学习笔记和资料。
*   **`java/`**: Java编程语言的核心知识、框架学习笔记和代码示例。
*   **`assets/计算机网络/`**: 计算机网络相关的笔记，可能包含图片等附件资源。
*   **`问题总结/`**: 在学习或工作中遇到的具体技术问题及其解决方案的记录。
*   **`fronted.md`**: 前端开发（如HTML, CSS, JavaScript, 框架等）的学习笔记。
*   **`question.md`**: 一些通用或待整理的技术问题汇总。

> **注意**：`.obsidian/` 目录是Obsidian软件的本地配置文件夹，它不影响笔记内容的阅读，但可以帮助您在本仓库中直接打开为Obsidian仓库。
