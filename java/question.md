# Java基础面试题

1. Java 有哪些核心特点？
	封装，继承，多态。封装是将一些属性和对象封装到一个类中，便于复用，聚合度更高。继承是子类可以通过继承复用父类的一些方法。多态分为编译时多态和运行时多态，编译时多态即函数重载，通过不同类型的参数可以定义多个同名函数，在调用时会根据参数的类型和数量来判断调用的是哪个方法；运行时多态是重载方法重写，子类可以重写父类的方法，在调用函数时会根据变量的类型判断调用的是子类还是父类的方法
2. Java 为什么能够实现跨平台？
	Java的一次编译，到处运行是因为Java在编译时是将代码编译成字节码，然后通过不同操作系统上的JVM虚拟机，编译成对应机器的机器码。然后再不同的操作系统中运行
3. JVM、JRE、JDK 分别是什么？它们之间有什么关系？
	JVM是Java虚拟机，Java程序能够一次编译，到处运行就是因为有JVM。JRE是JVM+Java程序运行的核心类库。JDK包括了JVM+Java程序运行的核心类库+Java程序所需的工具包
4. Java 为什么既可以说是编译型语言，又可以说是解释型语言？
	因为Java既可以通过javac编译成字节码，也有字节码在机器中通过JIT解释器逐句解释成机器码的功能。所以可以说Java即是编译型语言，又是解释型语言
5. Java 中参数传递为什么只有值传递？
	Java中参数传递如果传递的是基本类型，传递的就是值的副本，修改值时对原变量没有影响。传递的参数如果是对象，传递的就是对象的副本，修改时也对原对象无影响。但是如果通过对象内部的操作进行修改，就会影响原对象
6. Java 中基本数据类型有哪些？分别占多少字节？
	int 4B,long 8B,char 1B,double 8B,float 8B,boolean 1B
7. 自动类型转换和强制类型转换有什么区别？
	自动类型转换就是小范围的转换成大范围的，例如int型变量转换成long类型的变量，强制类型转换就是大范围的转换为小范围的，例如long类型变量转换成int类型的变量
	对于对象，子类转换成父类就是自动类型转换，可以直接通过父类变量来接受子类，父类转换成子类就是强制类型转换，必须要显式将变量转换成子类，然后才可以通过子类变量接收
8. 为什么金融场景中通常不建议使用 `float` 或 `double` 做精确计算？
	因为float和double都会有精度丢失的问题，做精确计算可能会导致计算错误，例如0.1+0.2不等于0.3之类的问题
9. `BigDecimal` 为什么推荐使用字符串构造？
	因为如果直接使用浮点类型的数值可能会有精度丢失的问题
10. 什么是自动装箱和自动拆箱？它们会带来什么问题？
	自动装箱就是int类型的变量可以自动转换成Integer类型的变量，从而可以进行一些Integer类的操作，自动拆箱就是Integer类型的变量可以自动转换成int类型的变量
	频繁的自动装箱会带来性能损耗的问题，并且在拆箱时可能会有null值带来的空指针异常
11. 为什么 Java 中需要 `Integer` 这样的包装类？
	因为Java中的泛型只能接收对象类型的变量，不能直接接收基本类型，并且包装类也提供了很多实用的方法，例如Integer.parseInt()方法
12. `int` 和 `Integer` 有哪些区别？
	int是基本类型，Integer是对象类型，int只占4B，只存储数据；Integer占用的内存更多，除了要存储数据之外，还需要存储对象头等一些类信息。Integer可以为null
13. 什么是 `Integer` 缓存？范围是多少？
	Java中提供了Integer缓存池，范围是-128~127，如果要使用这个范围内的Integer对象，可以直接复用缓存池中的对象，而不需要重新创建新的对象。但是如果通过new Integer创建，还是会创建新的对象
14. 面向对象的三大特性是什么？
	封装继承多态
